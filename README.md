### Instalação

2. Clone the repo
   ```sh
   git clone https://github.com/isaniovitor/CineFlix.git
   ```
3. Instale os packages
   ```sh
   yarn
   ```
4. Inicie o Projeto 
   ```js
   expo start
   ```
